Introduction

Using devices such as Jawbone Up, Nike FuelBand, and Fitbit it is now possible to collect a large amount of data about personal activity relatively inexpensively. These type of devices are part of the quantified self movement a group of enthusiasts who take measurements about themselves regularly to improve their health, to find patterns in their behavior, or because they are tech geeks. One thing that people regularly do is quantify how much of a particular activity they do, but they rarely quantify how well they do it.

The dataset used for this project is Weight Lifting Exercises Dataset1. The goal of this project is to use data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants and predict the manner in which they did the exercise. The prediction variable is classe in the training dataset.

Class A corresponds to the specified execution of the exercise, while the other 4 classes correspond to common mistakes. Participants were supervised by an experienced weight lifter to make sure the execution complied to the manner they were supposed to simulate. The exercises were performed by six male participants aged between 20-28 years, with little weight lifting experience. We made sure that all participants could easily simulate the mistakes in a safe and controlled manner by using a relatively light dumbbell (1.25kg).

Partitioning the data

Now that we have cleaned the data, lets partition the data into training and validation sets. Well keep 70% of the data for training and will use the rest of it for validation test.

inTrain <- createDataPartition(y=training$classe, p=0.7, list = FALSE)
trainData <- training[inTrain,]
validationData <- training[-inTrain,]

Model Selection

For classification problems, algorithms such as Support Vector Machines(SVMs), Random Forests, Gradient Boosted Trees, Decision Trees, K-Nearest Neighbors, Linear Regression etc tends to work pretty well. So, well try to model our data using these algorithms and compare their accuracies to select the best performing algorithm.

First, lets try using Support Vector Machines:

# Setting the training control parameter for 3-fold cross validation
ctrl <- trainControl(method = "cv", number = 3, verboseIter = FALSE)

# Training an SVM model
modelSVM <- train(classe ~ ., data=trainData, method="svmLinear", trControl = ctrl, tunelength = 5, verbose = FALSE)

# Predicting on validation data and checking for accuracy
predsSVM <- predict(modelSVM, validationData)
confusionMatrix(predsSVM, factor(validationData$classe))
